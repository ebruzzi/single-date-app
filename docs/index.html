This app automatically validates your cart to ensure only one delivery date variant can be ordered at a time.

How it works:

  • No configuration required
  • Works automatically after installation
  • Validates cart contents in real-time
  • Prevents checkout with multiple delivery dates

Note: This is a function extension app with no admin interface. The validation runs automatically in your checkout process.
